What is the OSI Model?

Open Systems Interconnection Model

The OSI Model (Open Systems Interconnection Model) is a conceptual framework used to describe the functions of a networking system. The OSI model characterizes computing functions into a universal set of rules and requirements in order to support interoperability between different products and software. In the OSI reference model, the communications between a computing system are split into seven different abstraction layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application.

Created at a time when network computing was in its infancy, the OSI was published in 1984 by the International Organization for Standardization (ISO). Though it does not always map directly to specific systems, the OSI Model is still used today as a means to describe Network Architecture. (1)

The 7 Layers

The 7 Layers of the OSI Model

The OSI Model uses a layered approach to transmit data from one computer to another. These 7 Layers are:

  1. Application
  2. Presentation
  3. Session
  4. Transport
  5. Network
  6. Data Link
  7. Physical

Encapsulation and De-Encapsulation

Data Encapsulation and De-Encapsulation in Networking

When Data is sent from one node to another in a computer network, that data is encapsulated at the sender's side, and then de-encapsulated at the reciever's end.

Data Encapsulation is the process which some extra information is added to the data item to add some features to it.

De-Encapsulation is the reverse process, where the encapsulated information is removed from the recieved data to obtain the original data. (2)

The OSI Model

The OSI Model (Open System Interconnection Reference Model) was created by the International Organization for Standardization (ISO) to help cocmputers communicate with computers made by different Manufacturers. It was meant to help vendors create interoperable network devices and software in the form of protocols or standards.

It is the primary architectural model for networkds and describes how data and network information are communicated from an application on one computer through the network media to an application on another computer. To do this, the OSI Model breaks things down into layers.
